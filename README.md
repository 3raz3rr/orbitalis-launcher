# Orbitalis Launcher

Клиентский лаунчер Orbitalis / Horny Jail.

В репозитории только клиент - исходники того, из чего собирается `.exe`. Серверная часть (новости, auth, CDN) отдельно.

**Скачать:** https://devassets.hornyjail.space/launcher/latest

**Исходники:** можно собрать самому (`npm run dist`) или сверить готовый файл с VirusTotal.

Discord: https://discord.gg/uj8HN4GB7s

## Зачем это

Не у всех нормально скачиваются серверные ресурсы. Причины разные: проблемы с провайдером, блокировки между странами, нестабильный BYOND asset transport / webroot. Лаунчер как раз чтобы обходить это проще (сама раздача ресурсов через лаунчер ещё не прикручена). Плюс не ждать условные 107 минут на каждые ~300 МБ.

Пока этой фичи нет, чтобы лаунчер не пустовал, прикручен просмотр своего аккаунта: заметки, часы, блокировки, персонажи с превью. Во многом поэтому его и выложили - многим хотелось такое о себе смотреть.

Также можно:

- быстро подключаться одной кнопкой;
- смотреть пинг до подключения и выбрать удобный маршрут (прямой / прокси);
- одной кнопкой пропатчить BYOND: обойти ~30 секунд рекламной задержки;
- снять диагностику, если по какой-то причине не пускает на сервер (проверка пути до сервера), и отправить её админам.

Сверху планируются инструменты для админов, возможно для разработчиков, и в целом всё полезное по дороге.

Есть страничка на itch. Есть люди, новые в SS13: для них вся инфра BYOND выглядит устаревшей и неочевидной (туда нажми, это скачай, тут найди). Поэтому в лаунчере есть кнопки на вики и на установку BYOND.

Вход в лаунчер через Discord (наш бот): аккаунты уже связаны в БД, без отдельных сайтов и регистраций.
